□導入方法
まずは本体全部を一旦別の場所へとコピペした後、
「ERB」→「SLG」→「艦船装備」フォルダ内にある
・「ENGIN」
・「MAIN_WEAPON」
・「SIELD」
・「SUB_WEAPON」
上記5つのフォルダを中身のファイルごと削除してから、
このパッチのERBを上書きしてください。
