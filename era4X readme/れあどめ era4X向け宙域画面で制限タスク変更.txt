readme era4X 宙域画面で制限タスク変更
【バリアント】era4X ver1.4.10

（対応環境）
 era4X ver1.4.10　ERBフォルダに上書きしてください


（このパッチは）
　宙域画面（船動かしてるとき）にインフォ画面を追加します
　同宙域にいる味方船へのタスク指示ができます。
　相変わらず乗員がワープできます仕様です。どうしましょうね？
　親愛嫁を難破船に送り込んで放置しなくてよくなります。[移動]or[追従]タスクしてね


===変更ファイル毎内容===
[\SLG\SLG_SHOP.ERB]
  宙域画面に [z]インフォ を追加します
[\SYSTEM\INFO_4x.ERB]
  タスク変更に制限を追加。あなたが星・宇宙港にいない場合は同宙域の味方にのみ指示が出せます
=================
era4Xにおける本パッチの改造・流用は制限しません
その他への利用はera4X本体の方針に従ってください
