readme era4X向け 船倉内表示パッチ
【バリアント】era4X ver1.4.11

（対応環境）
 (本体)era4X ver1.4.11
　　＋era4X向け艦隊表示手直し
　　＋era4X1411パッチ ｗａｓｄ移動の適用
　　＋era4X向けアイテムリファクタリング


（このパッチは）
　輸送船が船倉内に積んでるアイテムの表示をつけます
　港内メニュー及び宙域画面から呼び出す「インフォ」から船名をクリック

===変更ファイル毎内容===
[\SLG\SHIP_WINDOW.ERB]
  船倉表示を追加　@CREATE_WINDOW_SHIPINFO_CARGO
[\SYSTEM\WINDOW_SETUP.ERB]
　CARGO用にwindow7番を追加
=================
era4Xにおける本パッチの改造・流用は制限しません
その他への利用はera4X本体の方針に従ってください
