　→アレス宗教国家の独自艦艇です。

　FEAT：レールガン系主砲＆副砲
　→射程が長く、一発辺りの威力が大きい代わりに発射レートの少ない主砲と、
　　命中率が高い代わりにやや高価な単発の副砲です。どちらも実体弾属性。

　FEAT：小型ロケット
　→魚雷属性の武器です。命中率は更に低いものの、射程は２。
　　※もし中型ロケットや大型ロケットを作る場合、射程２に統一して下さい。

　FEAT：速射砲
　→ビーム副砲のコンパチみたいな性能の実体弾副砲です。

　FEAT：不規則航行(「偏差射撃」の従来効果)
