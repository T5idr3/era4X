era4X用コロニー建造物追加パッチ_バラック住居等
※本体に取り込まれた段階で諸々の権利は放棄します。


コロニー建造物について以下、３つを追加するパッチです。
(1)バラック住居…住宅数5を供給する建造物（通常の住居が3）。ただし一個ごとに満足度需要を+3する
(2)動物園…食料需要を+1して満足度を+2する建造物。
(3)送電集中管理局…コロニー全体の電力生産量を10%増やす建造物。コロニーひとつにつき一個のみ建造可能。POP毎に維持費がかかる(※)。

※(3)が建造されていると労働しているPOP1ごとに15000の維持費が税収から差し引かれます。
つまり、必要もないのに作ると大損こかされるだけです。現状は資源採掘のお供にするのが主要な用途と思われます。
