# エル口上

------




## 概要

　era4Xの賑やかしにあれこれ喋るキャラクターが欲しかったので試験的に口上を書いてみました。
「あなた」がアースにいる場合、奔放で飄々とした研究者志望の少女エルの加入クエストが出現します。

設定的には知略やアノマリー分析で強みを発揮できるようにしたいところですが、
今は概ね喋るだけです。
現時点ではキャラ造形から設定、パラメータ、スキルその他もろもろ設定周りの練りこみ不足が否めないですが、
それなりに賑やかしてはくれると思います。
era4Xはバリアントの特性上シチュエーションの幅が非常に広いので適宜加筆できたらいいですね。


※初版のみQUEST.ERBの修正パッチを同梱しています。詳細は下記を参照ください。


## 導入

　[必須]ERB、CSVのフォルダにそのまま上書きしてください。
　[任意]※２を参照

　※１．以下のファイルは上書きが発生します。
　SLG4X/QUEST.ERB
　SLG4X/QUEST.ERH 	

　※２．キャラクター描写の一部がSYSTEMフォルダ以下に定義されており元バリアントのままなので、
　SLGパートの一部の描写がなんとも小悪魔な感じになります。
　 気になる場合は同梱の「取込時のERB修正点.txt」を参照して修正してください。
　 このあたりの定義や服装の情報はeraALKにならってキャラクターのCSVに持たせられると弄りやすそう。



## 追加・変更点

### 2021/05/01
- エル口上(K7)、加入クエスト(QUEST_70)を作成。
  - 加入条件：「アース」の宇宙港にいる場合加入クエストが出現
- QUEST.ERBを微修正
  - クエスト選択時に(Y/N)で受領可否確認をするように
  - クエストファイルにクエスト概要が記述されている場合、受領前に概要を出力するように
    - QUEST_XX に関数@QUEST_DESCRIPTIONが定義されている場合、その内容を出力します。
- QUEST_70が読み込まれるようにQUEST.ERHのCONST QUEST_LENGTH を修正。

※クエスト番号は暫定的に70を割り当てさせてもいました。71,72...にサブクエストとかぶら下げられたらいいなぁと。
不都合があれば再採番していただきたく。
あるいはサブクエストをぶら下げられる余地があるよう、子番を設けてQUEST_XX_YY_"QUESTNAME"のような採番形式にするのもアリかもしれませんね。

## 改変・流用について

- eraに使用する場合に限り削除・修正・加筆・改変・二次配布・他キャラ・他バリアントへの移植可。良識におまかせします。
- era以外の利用不可

## キャラ造形に関する設定など

- ### プロフィール

  - 名前：エル・シュテルン
    - 設定上の正式なファーストネームはエレオノーラですがゲーム中表記及び呼称、ERB上の表記は基本的にエルで統一します。
  - 性別：女性
  - 年齢：14～17歳くらい
  - 国籍：民主主義国
  - 職業：元大学生（専門は宇宙物理学、他に情報工学に秀でる）


- ### 外見・身体的特徴・特記すべき容姿等

  - 身長：年齢標準
  - 髪型：ロング
  - 髪色：白
  - 肌の色：色白
  - 瞳の色：マリンブルー
    - 先祖が社会主義国の亡命者ため、容姿は北欧・ロシア系のイメージです。
  - 体型：細め
  - 3サイズ：胸が大変慎ましいこと以外は特に定めていないので想像と乱数におまかせします。


- ### 性格、口調、趣味、趣向、特技、政治的信条など

  - 一人称：私(ワタシ)　「あなた」への呼称：キミ　それ以外の呼称：設定上の呼称に準ず
  - 性格：自由奔放、楽天的、自信家、好奇心旺盛、実践主義者
  - 口調：「～だよ、～だね、かな、だろう、～しようじゃないか、～したまえ」
    - 自分自身の能力に対して一定の自負心があります。口調と相まって他者に尊大な印象を与える傾向がありますが、本人は無自覚です。
  - 趣味、特技：読書、研究、食事
    - とてもよくたべる。とても。
  - 信条など
    - 政治や国際情勢にはあまり関心がないようです。
    - フィールドワークを極めて重視する傾向があります。自分の手を動かさずにはいられないようです。
      - （特に練っていないが幼少期に著名な人物から影響を受けた……とかで関係性を膨らませたい）


- ### 略歴

  民主主義国の主星（アース？）で特段何の不自由なく育った少女。
  明晰な頭脳で勉学に非常に秀で、十代前半にて主星の名門工科大学に入学を認められる。
  大学入学後は研究室にこもり宇宙物理学の研究に勤しんでいたが、
  机上で、延々と数値とシミュレーターと格闘するだけの日々に嫌気が差し出奔を決意。
  順風満帆の学生生活を放り出して単身宇宙を目指す。
