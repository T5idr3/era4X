# ライブイベントパッチ

## 概要
+ 交流パートにライブイベントを模したミニゲーム[ライブ活動]を実行するコマンドを追加します
 - キャラクターのABLステータスとして演技力ＣＯ(COOL)、演技力ＣＵ(CUTE)、演技力ＰＡ(PASSION)、演技力ＬＥ(LEWD)を追加します
 - 歌唱と上記４つのステータスの総計、パフォーマンスによって獲得した補正値を基にスコアを算出します
 - ライブには上記の演技力に対応した4つの演目(COOL,CUTE,PASSION,LEWD)のいずれかが設定されます
 - いかがわしいライブ(演目:LEWD)での高スコア獲得はera的な結果を招きます
 - 条件を満たす場合健全なライブをダメな感じにすることも可能です

+ [ライブ活動]を解禁するクエストを追加
 - QUEST No.35「One Step To Broadway」民主主義国所有の星系港で受注可能に


## 条件など
+ [ライブ活動]はキャラクターは歌唱40以上のキャラクターのみ実行可能
    

## ルールに関する補足
- パフォーマンス値:歌唱と４種の演技力に対して、対応する補正値を掛けて算出されるスコア
- 心理抵抗値:観衆が演技者に対して抱いている心の壁や演技に対する期待感を示す値
- エモーション値：(パフォーマンス値 - 心理抵抗値)で算出。キャラクターの演技がどれだけ観衆に感銘を与えたかを示す。エモみ。この値が高いほど高評価となる
