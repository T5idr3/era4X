【導入方法】
SHIP_MODEL.ERHが競合している為、このパッチを導入後に
製作スレ4の>>841「4X用航巡+装備更新＋特殊分類混合パッチ」を導入して下さい。
逆の場合、トーン級航空巡洋艦と天津級重航空巡洋艦が認識されません。

　追加した物品の元ネタとか設定とかデザインとかの解説
～～～
【ポーン級機動甲冑】
　コメントアウトでジェインが愚痴っている通り、元ネタは絢爛舞踏祭の士翼号。
　世界観的な一応の設計コンセプトは生残性を高めた高コストな人型魚雷艇で、
　妙に回避が高いのは遮断シールドのステルス効果という設定。
　外観イメージとしては球体フレームに士翼号みたいな流線形の手足が付いていて、
　その球体フレームは分割式で頭や下半身として一部を旋回可能。
　設定上、友愛騎士団などでそれぞれ3～4機ぐらい配備されていることにしている。


【機動甲冑『オーヴァス』】
　やはり元ネタは絢爛舞踏祭の希望号。
　外観イメージはポーン級より少々ヒロイック且つ全体的に太くなっている。
　しかし顔はポーン級と同じく（・ω・）である。
　扱いとしてはイベントで1機だけ入手するかNPC専用にするのが好ましい。


【マンティスⅣ機動歩兵】
　フェイツィⅫ機動歩兵を見て思い付きで作成。元ネタは勿論、クリスタルゴ。
　元ネタの「光学兵器に強い」という設定をシールド持ちとして再現。


【エグゾスケルトン通常型、砲戦仕様、近接仕様】
　元ネタはガンパレード・マーチより、それぞれ可憐通常型、Ｄ型、本国仕様。
　当然ながらデザイン的には補助腕付きで、装甲宇宙服などの上から着用する想定。
