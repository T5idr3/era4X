era4X用船体追加パッチ_戦闘艇
※本体に取り込まれた段階で諸々の権利は放棄します。

購入可能な艦艇に、戦闘艇と称する独自艦艇を追加します。

空母に搭載するような小型の単座式戦闘艇ではなく、
コルベットに近いサイズの大型単座式戦闘艇です。
コルベットよりも断然お安く、なのに攻撃力はほぼ同等。
しかし、副砲がありません。装甲がありません。オプションがありません。
ものによっては、シールドすらありません。
機動性も大きく下がっているので回避も期待できません。

初期装備のコルベットとかフリゲートあたりが相手なら
しっかり艤装してやれば蹴散らせますが、
敵に長距離砲撃してくる艦が現れた瞬間、粉みじんになります。
その頃までには卒業するか、それでも有効に使える手段を見出しましょう。

以下のものがあるので、駆け出し傭兵の皆さんはお近くの造船所にどうぞ。

・銀河人民統一連邦以外ならどこででも買える戦闘艇
・企業連合独自規格の戦闘艇
・皇国独自規格の戦闘艇
・機械文明独自規格の戦闘艇
・銀河人民統一連邦独自規格で、ちょっぴり信用を得ないと買えない戦闘艇
