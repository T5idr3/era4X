era4X用施設とコマンド追加パッチ_撮影設備および静画を撮る等再調整バグ修正
※本体に取り込まれた段階で諸々の権利は放棄します。

当パッチは以下を実施しています。
・日常系の新規コマンド、静画を撮るを追加したパッチの修正。
　→見せ系コマンド（例えば胸見せ）を使った本人以外を撮ってもその状態（胸見せ）が適用されていた不具合を、ポーズの継続コマンド化によって修正
・胸見せ、尻見せ、股見せコマンドを継続可能化、および再調整。主に射精感を下方修正
・居住区のルーム『高電力娯楽室』を追加
　→４つまでモジュール設置可能な部屋です。
　→専用の電源が必要な大型設備（たとえば映画館とかライブのハコとか）を配置するのを想定しています。
・居住区のモジュール『撮影設備』を追加
　→『高電力娯楽室』にのみ設置可能
　→定員９名、遮蔽、自動施錠属性持ち
・居住区コマンド『水着姿になる』『裸になる』を追加
　→継続コマンド。胸見せ、尻見せ、股見せ、静画を撮るの効果を強化します。具体的には『遊泳場』、『温泉』でデート中に同コマンドを使ったのと同じ状態に。
　→現状では、居住区モジュール『撮影設備』でのみ使用可能です。
・居住区コマンド『気分のまま愉しむ』を追加
　→同行者全員の欲情が9以上になっていて、かつ『自動施錠』『遮蔽』属性持ちのモジュールで使用可能になるコマンドです。
　→その場の空気に流されて乱交開始、的なコマンドです。合意はつきません。

不具合を指摘いただいてから、および施設追加の許可をいただいてから
相当に長く間を空けることになってしまいました。申し訳ございません。




---------- 以下、『静画を撮る』公開時のreadme ----------


era4X用コマンド追加パッチ_静画を撮る＋胸見せなど再調整
※本体に取り込まれた段階で諸々の権利は放棄します。


日常系の新規コマンド、静画を撮るを追加しています。
普通に使うとスキンシップより弱い程度の効果しかなく、
撮影経験がつくしか特色のないコマンドですが…
胸見せ、尻見せ、股見せを実行した直後にこれを使うとエロポーズの撮影と化します。
遊泳場でなら水着姿の撮影ですし、温泉なら裸の撮影です。
水着姿は常識の範疇なんで、ポーズをとらせなければ普通の撮影ですが、
裸の撮影は当然エロです。エロを撮ると撮影側は射精感をもよおします。
居住区だと、撮る側の数は無制限、撮られる側は３人までです。

かなり突貫工事気味で作ったため、不具合潜在の可能性ありです。
見つかり次第、修正したものを上げようと思います。

また、胸見せなどの見せ系コマンドを再調整しました。
なずなとかが、どれだけ仲良くなってもエロポーズをとらせるだけで激怒するような状態は
調教ゲームとして考え物だ、と思ったため、歓楽ソースを増やし、反感ソースを大幅に下げました。
これで少なくとも、ある程度仲良くさえあれば上記のエロ撮影一回くらいは我慢してくれます。
また、実行条件にてマイナスが重なりまくるキャラだと、見せてもらうまで遠いにも限度があるわ、という状態だったため、
欲望レベル、露出レベルが上がることでも実行条件にプラスになるようにしました。これでだいぶマシになったはず。

何度もパラメータを乱高下させて申し訳ございません。
