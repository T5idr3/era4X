# 艦船データ入替え機能追加パッチ

## 概要
+ 宙域画面の「[l]艦船一覧」に艦船データ(SHIP_ID)を入替える機能「[c]入替モード」を追加します
 - 艦船データを入替える性質上、不具合があった場合艦船データに不整合がおきてしまうためバックアップの取得を強く推奨します
+ 開発者の方向け
 - 艦船データは現状各要素の下にSHIP_IDがぶら下がっている構造になっているため、新たにSHIP_IDに紐づく要素を追加した場合は併せて「lib_MAP.ERB」の@SWAP_SHIPDATAも修正をお願いします


## 導入
- ERBに上書きしてください

## ライセンスについて
- era4Xのライセンスを継承します
