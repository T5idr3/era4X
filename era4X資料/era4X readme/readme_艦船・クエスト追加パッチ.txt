# 艦船・クエスト追加パッチ
------

## 概要
+ オプション装備枠を２つ持つ艦船を民主主義国と企業連合に追加

+ 手動でのアイテムの売買に関しては積載量枠があればどの艦種でも可能に
 - 交易、輸送は変更なし。輸送船のみ
 - 増設コンテナの艦種装備制限を撤廃
 - オプションをスロット化。SHIP_OPTION_SLOT　でスロット数の指定が可能に

+ eraで4Xな感じのクエスト、「蠢く星」を追加
 - イベント側でシチュエーションをある程度制御できるよう処理、フラグを追加

## 条件など
+ アンビシャス級試験艦
	巡洋艦　企業連合の星系港にて建造可能に
+ フリーダム級輸送船	
	輸送船　民主主義国家の星系港にて建造可能に
+ クエスト「蠢く星」
	社会主義国の星系港で受注可能に
    

## その他、補足等
ソロ艦での活動をある程度拡張することを企図して修正を加えています
追加艦は汎用艦を基準にステータスを振りましたが、バランスについてはちょっと自信がないです……
