 内容
---
①艦名候補の追加
　幾つかの陣営の艦名候補が（再び）増加します。
　入りきれなかったものは他艦種等へズラしています。
　かつて実在した艦艇由来の名称や、
　艦名にされてない実在人物由来の名称も少なからず使用しているので注意。
　あと、皇国は共通設定との兼ね合いから一部をガッツリ差し替えたので注意。

②スキル追加
　特定属性の武装を強化するスキルや、自艦の迎撃性能を強化するスキル等を追加。
　強すぎる又は弱すぎる等、バランス的に問題があれば修正しても構いません。
　勿論、構文が間違っている場合も修正して構いません。

③属性名の変更
　「戦闘機」となっていた属性名を「艦載機」にしました。

④自作キャラ関連
　一部素質の差し替え、成長型の変更、目星の追加を行いました。
