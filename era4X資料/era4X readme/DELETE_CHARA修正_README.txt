@DELETE_CHARA
NO:登録番号を使用する必要がある際にキャラクターの登録番号そのものを使っていた部分を修正
ランダムキャラと子供とカスタムキャラ等をより厳密に識別する様に
	(ランダムキャラを消してNOを詰めたら子供がランダムキャラの範囲に入ったり・カスタムキャラ等が子供の範囲に入る事になる処理であった)
EXISTCSVの返り値を判定に使う部分の修正
武将カウントが正常にカウントされる様に修正

@ARRANGEMENT_NO
呼び出すたびに「現在存在する武将の人数」分だけ加算する代わりに
汎用武将カウントに「現在存在する武将の人数」を代入するように
((実際に周回するところまでいかずとも)周回画面を開くたびに@ARRANGEMENT_NOが実行されるので汎用武将カウントが不正な値になる)
