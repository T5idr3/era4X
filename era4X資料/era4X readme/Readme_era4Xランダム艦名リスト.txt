◆対応環境
era4X ver1.4.9
ERBフォルダに上書きしてください

◆内容
・ステ〇リスっぽい艦船用ランダムネームシステムを追加します。
・INIT_SHIPもしくはINIT_PRESET_SHIPで艦船を作成する際に、ARGS:0を指定しないことで、
ネームリストから使用回数が少ない名前を自動で選択します。

◆ネームリストの追加方法
1) 添付のテンプレートを使用して、NAME_LIST:ID:（艦種）:0 = の右側にカッコいい名前を埋めます。
2) ネームリスト名は、ニューゲーム時にプレイヤーが使用するネームリストを選択する際に表示されます。カッコいい名前をつけてください。
3) @GET_NAME_OF_SHIP_NAME_LIST_X()と@REGISTER_SHIP_NAME_LIST_X(ID, NAME_LIST)の「X」を未使用のIDで置き換えます。使用中のIDはNAME_LIST.ERHの中で、「#DIM CONST 艦名_デフォルト = 0」のような形式で書かれている値です。MAX_NAME_LIST未満で未使用の数字を選択ください。
（最大値は仮に勢力数としていますが、足りなければ増やしてかまいません。）
4) できたてのネームリストをERBフォルダにぶちこみます。これだけでもニューゲーム時の候補に追加されます。
5) NPC勢力に作成したネームリストを使用させたい場合は、NAME_LIST.ERBの@INIT_SHIP_NAME_LIST関数の中身を変更します。
CALL REGISTER_COUNTRYS_SHIP_NAME_LIST(国家ID, ネームリストID)
の引数で使用させたい国家と作成したネームリストのＩＤを指定してください。

◆その他
・艦名の候補は各ネームリストで艦種ごとに20個です。足りない場合は定数MAX_RANDOM_NAMEを変更してください。
・現状のネームリストは趣味が駄々洩れ（デモクラティック・クルセイダーvsスペースソビエト）だったり全然埋まってなかったりするので、遠慮なく容赦なく変更ください！
