# 大破フラグ実装パッチ

## 概要
+ 大破フラグを新設
 - 鹵獲艦は大破扱いでの入手になります
 - プレイヤー陣営の艦が戦闘で撃破された際、即座にロストせず大破扱いとなります
 - 大破扱いの艦は戦闘には参加するものの攻撃は行なえません(輸送艦等の非戦闘艦と同じふるまいをします)
 - 大破艦は港で修復可能です(艦に設定されてる建造費の70%がコストとして必要)
 - 大破艦が戦闘で撃破された場合ロストします
 - アノマリー「漂う巡洋艦」は大破扱いでの入手になります


## 導入
- ERBに上書きしてください

- era4Xのライセンスを継承します
