era4X用裏施設追加パッチ_しゃせいかい
※本体に取り込まれた段階で諸々の権利は放棄します。


裏施設『しゃせいかい』を追加します。
ぶっちゃけると、女の子一人によってたかってぶっかける裏施設です。
本番はありません。男から女に触るのは全面的に禁止です。
以下のモードを搭載してます。

(1)モブ少女が、あなたを含む男４人の竿で遊ぶモード
(2)同行者が、モブ男４人の竿で遊ぶモード
(3)同行者が、男どもにひたすらぶっかけられまくるモード
(4)あなた自身が、男どもにひたすらぶっかけられまくるモード
(5)モブ少女が、男どもにひたすらぶっかけられまくるモード


裏施設『しゃせいかい』にアクセスするには、
星つきの宇宙港でクエスト『とある宇宙船乗りの廃業』を受け、
$500000を支払って入場パスを買う必要があります。
買ったら、星つきの宇宙港であれば以降どこででもアクセス可能です。

下着姿になるうんぬんテキストに書いてますが、
現状実態が伴っていません。フレーバーです。


※特に注意が必要な事項
『COM_AUTOSELECT_3_カテゴリの優先度設定リスト.ERB』に編集を加えています。
『なすがまま』にてＰ自慰を実行するため、582～584行目の記述を削除しています。
当該記述は、読む限り『特定の慰安モブ以外はＰ自慰を行わない』意図のもので、


『犬豚馬猿でなければ可。
　でなければ、慰安参加者であれば可。
　でなければ、特殊能力素質がサキュバス、自警団でなければ可。
　上すべてに該当なくば否』
にて『可』である

…かつ、

『該当の慰安モブが露出狂ではない』
にて『可』である

であれば、Ｐ自慰の実行優先度は-1。絶対に行わない


というものなので、『慰安_キャラ_露出狂』以外では絶対にＰ自慰は実行されない、
かつ、記述の登場人物がほぼeraTohoKに由来しているため、影響はほぼないと判断しました。
