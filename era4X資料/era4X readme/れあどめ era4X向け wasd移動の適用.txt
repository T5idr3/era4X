readme era4X向け wasd移動の適用
【バリアント】era4X ver1.4.11

（対応環境）
 (本体)era4X ver1.4.11
　＋era4X向け艦隊表示手直し
  上記パッチ適用後に上書き

（このパッチは）
　移動の際にキーボード入力[wasd]による操作が可能になります


===変更ファイル毎内容===
[\SLG\MAP_WINDOW.ERB]
  宙域画面用移動キーの調整、割り当て変更。高速／低速モード切替追加
[\SLG\SLG_SHOP.ERB]
　宙域画面入力を ONEINPUTSに変更、合わせて移動キーの変更
[emuera.config]
 「ONEINPUT系命令でマウスによる2文字以上の入力を許可する」 NO → YES
[\資料\era4X資料era4X フラグ使用状況一覧.txt]
　追加　FLAG:502　低速移動フラグ
=================
era4Xにおける本パッチの改造・流用は制限しません
その他への利用はera4X本体の方針に従ってください
