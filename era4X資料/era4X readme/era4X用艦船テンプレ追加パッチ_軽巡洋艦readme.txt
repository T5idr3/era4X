新規開発艦船に、軽巡洋艦のテンプレートを追加するパッチです。
巡洋艦のダウンサイジング版で、艦隊許容量は６。
シールド、装甲のサイズが３であることから防御力はほぼ巡洋艦に匹敵するものがありますが、
主砲と副砲はサイズ２。駆逐艦程度の火力しか出せません。
しかしながら回避力は１０あり、パラメータの伸ばし方によっては重雷装艦としてメシを食う余地があります。
巡洋艦になりきることはできませんが、それに匹敵する防御力を持った駆逐艦として振る舞うことは可能、といったテンプレートです。
数を揃えて戦列を組めば鉄壁になりえますが、そうするにはそろそろ艦隊許容量が少し重い。期待する役割に応じて駆逐艦と使い分けるといいです。
また、小艦隊の旗艦にもオススメ。
