era4X用ライブ活動追加カード１１枚
※本体に取り込まれた段階で諸々の権利は放棄します。

内容は以下。
・投げキッス
心理抵抗を下げつつCUTEを上げる、素で割と強力めなカードです。
わずかながらLEWDも上がりますが、一般ライブで怒られることはありません。

・ポージング系
CHAINの〆に使うと効果的なカード群です。
COOL、CUTE、PASSION。弱カード、強カードの組み合わせで計６枚あります。
弱カードは無いよりいいかな程度ですが、強カードだと露骨に狙う価値がある…
というバランスを想定して作っています。
強カードは対応する演技力７０から解放されます。

・セクシーポージング系
ポージング系の弱カードに近いですが、LEWDを上げられます。
一般ライブで使用しても怒られません。
社会通念にギリギリ反しない範囲でエロチックなポーズをとってて、
オカズにしようと思えば十分なるっていう想定です。いわゆるグレーゾーン。
LEWDのライブでもある程度役に立つと思われます。

・ドスケベポージング
良い子のみんなが見ちゃダメなポーズです。
一般ライブだとマジギレされます。
CHAINの〆に使うとLEWDを効果的に上げられるでしょう。
心理抵抗も下げるのでCHAINにせずとも使いどころはあるかと。
