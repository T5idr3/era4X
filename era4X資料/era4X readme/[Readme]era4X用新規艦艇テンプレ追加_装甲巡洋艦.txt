艦船設計の新規テンプレートとして「装甲巡洋艦」を追加するパッチです。
本体に取り込まれた段階で諸々の権利は放棄します。

装甲巡洋艦
・火力と引き換えに耐久力を向上させた巡洋艦です。艦隊の最前線で壁にする運用を想定しています。
　巡洋艦と比較するとシールドと装甲のスロット数が増加していますが、
　代わりに副砲サイズが3→1と大幅に低下し、スロット数も主砲、副砲ともに半分以下に減少しています。

[性能](カッコ内は巡洋艦テンプレの数値)
・編成コスト：8（8）
・耐久：6000（5000）
・副砲サイズ：1（3）
・主砲スロット数：1（2）
・副砲スロット数：1（3）
・シールドスロット数：4（2）
・装甲スロット数：3（2）
