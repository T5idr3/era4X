IS_VARIANT関係の可読性向上+船の特殊分類表示追加パッチ

◇パッチの目的
IS_VARIANTやそれを利用する際にマジックナンバーでの表記を使用せずに書ける様にして可読性を高めたい。
通常ゲームプレイの範疇でどの船がどの特殊分類を持っているのかを認識できる様にしたい。

◇ 具体的な変更点
SLG.ERHにIS_VARIANTの設定値の定数を追加

#DIM CONST 特殊分類_通常艦艇 = 0
#DIM CONST 特殊分類_メック = 1
#DIM CONST 特殊分類_生体艦 = 2
#DIM CONST 特殊分類_宇宙生物 = 3

同時更新の航空巡洋艦2隻のIS_VARIANTも同様の書き方に。

文字列定数として「SHIP_VARIANT_NAME」に各特殊分類の名称を設定。
(SHIP_VARIANT_NAME:0で「通常艦艇」からSHIP_VARIANT_NAME:3で「宇宙生物」まで設定済み)

@SHOW_SHIP_INFOと@SHIP_STATUS_DISPLAYに特殊分類の表示を追加
(「どの船がどの特殊分類を持っているのか」が分からないと今後スキル等が実装されても「どの船に適用できるのかが分からない」という問題を抱える事になります)

◇注意点
設定値の定数はあくまで可読性を向上させる手段なので、4以降を設定した場合に「#DIM CONST 特殊分類_〇〇 = 4」以降を設定しなくても機能はしますが
特殊分類の名称が追加されないと建艦時や宇宙港で船の情報を表示する際にエラーが発生して進行不能になります。
(文字列定数の配列を使って特殊分類名を表示するのに使用しているので配列外の数値を指定されるとエラーになります)
