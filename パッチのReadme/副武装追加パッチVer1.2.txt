-------------------------------------------------------------------------------------------------
	概要
-------------------------------------------------------------------------------------------------
era4Xに3種3TIERの副兵装を追加するパッチです。
パッチ詰め合わせに同封している作業機追加パッチとは競合せず、同時に使用することが可能です。

-------------------------------------------------------------------------------------------------
	使い方
-------------------------------------------------------------------------------------------------
そのままERBフォルダに上書きしてください。

-------------------------------------------------------------------------------------------------
	追加した物
-------------------------------------------------------------------------------------------------
副兵装：計9種
■TIER1クラスターレーザー
■TIER2クラスターレーザー
■TIER3クラスターレーザー
■TIER1ポーキュバイン
■TIER2ポーキュバイン
■TIER3ポーキュバイン
■TIER1補助小型魚雷
■TIER2補助小型魚雷
■TIER3補助小型魚雷

-------------------------------------------------------------------------------------------------
	更新部分
-------------------------------------------------------------------------------------------------
Ver1.1

■TIER1ポーキュバイン
■TIER2ポーキュバイン
■TIER3ポーキュバイン

EQUIP_UPDATE.ERBの
FOR ITER, 0, 40
を暫定的にFOR ITER, 0, 80に変更して対応。
