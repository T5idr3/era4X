# eratohoK ver1.25.7用 諜報パッチ v1.00

## 概要
自隣接していない敵都市の守備兵、駐屯部隊の兵数を報告させる諜報が使えるようになります。
諜報が可能な都市を選択すると、技術研究などのボタンに並んで諜報ボタンが表示されているので、そこから選択してください。


## 条件
諜報コマンドが使えるのは以下の条件の時です。
- 自都市と隣接していない
- 諜報を実行していない都市である
- いずれかの勢力が占領している都市である

## 著作権等
常識の範囲内で自由にしていただいて大丈夫です。

## バグ報告・要望
スレにて報告していただけると助かります。
