	変更部分
-------------------------------------------------------------------------------------------------
QUEST_24_模擬戦闘の敵艦隊の武装が大きく改善され、本格的な腕試しが可能になります。
さらに、依頼を受けるたびに敵艦隊の装備傾向がランダムに変更され、これといった必勝法が無くなっています。



-------------------------------------------------------------------------------------------------
	Q＆A
-------------------------------------------------------------------------------------------------
Q：製作理由は？
A：空母以外が揃いも揃ってTEAR1コイルガン一本で特攻してくる所為で、大型艦が船体性能を生かせずに沈んでいくだけの現状があまりにも見ていられなかったんで…
…という理由は建前で、新しいクエスト用に用意していた関数のテスト版という意味合いが強いです。

Q：なんか皇国っぽくない装備が現れるんですが
A：皇国軍による、近隣勢力のドクトリン研究の一環です。(機械文明は艦隊編成が独特すぎて再現不能だったため諦めた)
(かつて煮え湯を飲まされたハースバルズ王国風構成や、激しく敵対しているパロス教団風構成に勝った場合は、報酬に多めに色が着きます)

Q：これ強くなりすぎてない？
A：負けても修理代がタダで、倒した分だけお金がもらえるクエストなので多分大丈夫かと…

Q：主任！？貴様、(次の制作予定は)何をしようとしている！？
A：いやいやｗちょっとｗ敵がかなりランダムなクエストの作成を、ね！(核攻撃)
