別星系から逃げのびてきた亡国のお姫様
おしとやかで清楚なTHEお姫様といった感じの性格で１６歳くらいの少女
一国の姫として育てられたので政治や社交の能力が高い
心優しく誰かを助けられ救う力があるのならそれを行うべきという信念を持っている
趣味の料理が得意でめちゃデカイおっぱいにデカイ尻にむっつりスケベと色んな意味で都合の良すぎる女
申し訳程度の弱点として白兵と操艦が低く音痴
本人は直感や勘が鋭い程度にしか思ってないが能力こそ低いが無自覚の超能力者だったりする


かつては別星系を支配する王国のお姫様だったが諸侯貴族から下剋上をされ自分以外の一族郎党全員処刑された

父親である国王は支配領域拡大のため他星系に攻め込むべく軍拡を際限なく行いそのせいで民衆が貧窮に陥った
三年前に諸侯貴族が暴君打倒と民衆を煽り王は討たれその一族も全員処刑
そんな中第一王女でもあったティアは１０歳のころから貧困にあえぐ国民に慈善活動を行っていたこともあり
自身の周辺人物から好かれていたため周りの人間の尽力によりプレイヤーのいる宙域まで逃げ延びることが出来た
だがティアが生存していることを嗅ぎつけた彼女の存在が疎ましい者、利用したい者と様々な勢力の追手が迫ってきていた
そんな彼女を助けるかどうかはあなた次第といったところ
