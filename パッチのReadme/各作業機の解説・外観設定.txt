
◆デボラ級輸送船・ギデオン級作業船・モーセ級探査船

悪い意味で無個性なトリオ。少数精鋭ドクトリンの影響を受けているため性能こそ高いが、用途を考えるとお高いのがネック。

外見イメージはEndless SkyのHaulerⅠ級とNest級。残念ながらHaulerⅡ級やHaulerⅢ級みたいなコンテナ多段連結型は今のところ作るつもりなし。
原作では輸送艦となんちゃって大型空母で共用されたが、民主主義国家では民間船三種で使い回し。

◆URL-48M型作業船
基本性能重視型。
初期からそこそこの耐久性があるが、武装がよろしくない事から伸びしろはほぼ無い。

外見イメージはシンプルな中央フレームに各パーツをベタ付けした内部機構丸見えタイプ。
一応燃料タンクなどは後付けで装甲版が取り付けられているが、それでも流れ弾が当たって爆散する事案が絶えない。

◆マルチパーパス2級作業船

漢の中の漢のための白兵戦特化型。耐久が死んでいるため気合で避けてください。

外見イメージは船体正面及び側面から作業用アームがごってり生えた非常にシンプルなスタイル。
気になる作業用アームの数は20本を想定。明らかにAIの制御を前提とした設計といえる。
アームを8本ずつ受け持つ専属パイロット席が2つ付けられたとはいえ、それでもまだ人間には多すぎる数なのは疑いようがない。

余談として、解説文でちょっと触れられている前級の『マルチパーパス級』は16本のアームの操縦と操船を船長一人でやる事を要求していた。

武装構成は某人物を参考にしているが、癇癪を起して部下に当たり散らしたり「ケノービ！」等と喚いたりはしない。

◆ヘイロタイ型作業機

どうしてこうなった型お笑い機体。
性能はネタに片足…どころか頭から突っ込んでいるが汎用作業艇より安い。

作った後で気付いたが、扱い的にはどっかの丸い棺桶そのもの。
ただしキャノンが無い代わりに戦闘用機動歩兵とガップリ４つで組み合えるぐらいの強度は保証されている。
ちなみに、平均より上の地位の出身の人間は基本的に汎用型作業船に乗っており、この機体に乗ることはない程度の扱いである。

外見イメージは作業用に改造されたツインラッド。『陸上での運用が可能』という一文の結果がこれ。当然ながら宙域性能は犠牲になった。
内側の空洞部は前半分が建築資材や採掘した資源を載せるスペースとなっており、
後ろ半分はメインスラスターや作業用アームの接続部となっている。
コクピットは左右の各作業用アーム群の中央に位置しており、最低限の保護用フレーム以外は何もない野ざらし仕様。この影響で無重力下では宇宙服の着用が必須。
ちなみに陸上での走行速度は遅く、同サイズ相手への地ならし攻撃はほぼ意味がない。

◆EF・コール級作業機

潜在能力重視型。
初期性能は耐久不足が原因でパッとしないが、武装の質がそこそこで装甲とシールドを換装すればかなり戦える性能になる。

後継機(ガワと船種だけ作業船の戦闘用機動歩兵)も予定しているものの、
制作者である開発主任の設定共々案が寝れていないためとりあえず今回は見送り。

外見イメージは下半身がブースターになり、左腕がガストーチ内臓武器腕に取り換えられて装甲が超ダサい感じになったデナン・ゾン。
有線爆弾や大型機材、資材運搬用コンテナは空いた背部や腰にマウントし、
後付け可能なシールドジェネレータは右肩にダブルオーのGNドライブ風に取り付けられる。
